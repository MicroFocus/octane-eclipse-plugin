This is an update site for the Eclipse IDE Integration for ALM Octane, version 16.0.400.v20220509_50.

Eclipse Marketplace Install

Drag to your running Eclipse workspace (Requires Eclipse Marketplace Client)

Alternative Install

Go to Help > Install New Software, and paste this URL into the dialog box. Follow the on-screen instructions from there.
https://microfocus.github.io/octane-eclipse-plugin/update-site

Offline Install

You can download a zip archive of this build by clicking the following link: 16.0.400.v20220509_50.zip

To install from a zip archive on a machine with no network access to this update site:
Go to Help > Install New Software
In the "Work with" text field add the local path of the zip archive in the following format: jar:file:/C:/Download/16.0.400.v20220509_50.zip!/
As an alternative you can click "Add" > "Archive" and locate the zip file on the disk.
